Seria's dotfiles!

## Fish

- Don't add `reboot` into shell history
- Ctrl+backspace deletes entire words

## fcitx5

- Win+Space to switch between English, Traditional Chinese, and Japanese
- Traditional Chinese uses the Hsu keyboard

## Sublime Text

- Alt+Shift+F to prettify a JSON file
- Discard files automatically on exit
- Always open a new fresh window on launch
